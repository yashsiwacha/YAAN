# Desktop App Icons

Place your application icons here:

## Required Icons

1. **icon.png** (512x512 or 1024x1024)
   - Main application icon
   - Used in window title bar and taskbar
   - Should be PNG format with transparency

2. **tray-icon.png** (16x16 to 32x32)
   - System tray icon
   - Should be simple, monochrome design for best visibility
   - Works on light and dark system themes

3. **icon.ico** (Windows) - Optional
   - Multi-resolution Windows icon
   - Generated by electron-builder if not provided

4. **icon.icns** (macOS) - Optional
   - macOS application bundle icon
   - Generated by electron-builder if not provided

## Icon Generation

You can use electron-builder to auto-generate platform-specific icons from a single PNG:

```bash
# Install icon generator
npm install --save-dev electron-icon-builder

# Generate from a single 1024x1024 PNG
npx electron-icon-builder --input=./icon-source.png --output=./assets
```

## Design Guidelines

- **Simple & Recognizable:** Icon should be clear at small sizes
- **Brand Colors:** Use your brand colors consistently
- **Transparency:** PNG icons should have transparent backgrounds
- **High Resolution:** Start with at least 512x512px for scaling
